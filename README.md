# Dine-in Restaurant Website

## 📖 Overview

A responsive restaurant website built as part of the **Frontend Mentor Dine Restaurant challenge**.

The project focuses on building modern UI with clean structure, responsive layout, and interactive elements using vanilla JavaScript.

- Home page showcasing restaurant highlights and event sections
- Booking page with a reservation form and validation

---

## 🚧 Project Status

**In Active Development**

---

## 🛠️ Tech Stack

- HTML5 (Semantic markup)
- CSS3 (Flexbox, Grid, Responsive Design)
- Vanilla JavaScript (DOM manipulation, form validation, UI interactions)

## 📱 Key Features

### Navigation & Layout

- Multi-page navigation (Home / Booking)
- Fully responsive design (mobile, tablet, desktop)

### Dynamic Content

- Interactive event sections:
  - Family Gatherings
  - Special Events
  - Social Events

### Booking System

- Reservation form UI
- Input validation:
  - Required fields (Name, Email)
  - Email format validation
  - Date & time selection handling
- Hover & focus states for accessibility
